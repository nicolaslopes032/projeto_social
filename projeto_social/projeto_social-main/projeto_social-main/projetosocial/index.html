Biblioteca Login

Seja bem-vindo a Multimeios online, o site perfeito para o aluno leitor da rede estadual !

Como funciona a plataforma:

O sistema tem como objetivo armazenar e gerenciar os livros da Biblioteca de uma escola de forma online, permitindo verificar se um livro está disponível, emprestado ou indisponível, sem a necessidade de procurar fisicamente. Nosso sistema ajudará todos os alunos que têm o hábito de ler com livros da biblioteca da escola.
